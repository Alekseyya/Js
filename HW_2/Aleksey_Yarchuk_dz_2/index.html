Дан код:

var a = 1, b = 1, c, d;
c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

Ответ

c = ++a; alert(c); - возвратит 2 по тому, что в префиксной форме
инкрементирование просходит сразу, потом возвращает значение.
d = b++; alert(d); - возвратит 1, т.к сначала происходит возврашение
значения, а потом уже инкрементирование. c = (2+ ++a); alert(c); - выведет 5, т.к a = 2 дальше выполнится унарный
оператор a=3, дальше выполнится сложение 2 + 3 =5 d = (2+ b++); alert(d); - b = 2, т.к оно возвратило значение, оно не будет 3
значит 2 + 2 =4 В итоге a = 3 т.к выполнилось две префиксные формы b = 3 в последнем алерте уже возвратило инкрементированное значение.
